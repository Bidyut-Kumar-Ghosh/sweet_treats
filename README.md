# Sweet Treats Bakery Website

## Overview

Sweet Treats Bakery is a dynamic bakery management website developed using PHP, MySQL, HTML, CSS, and JavaScript. The system allows customers to browse bakery products, view daily menu updates, and submit feedback. An administrator dashboard is also included for menu management and feedback monitoring.

---

# Features

- Responsive homepage with image slider
- Dynamic daily bakery menu
- Customer feedback system
- Search functionality for menu items
- Administrator login system
- Add/Delete bakery menu items
- View customer feedback
- Session-based authentication
- Mobile responsive navigation menu
